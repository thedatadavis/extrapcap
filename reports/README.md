# Research reports

Generated reports belong here and should record the data snapshot, code version, assumptions, variant, and whether option prices are reconstructed or observed. The checked-in `sample-backtest.json` is only a smoke artifact; it is not evidence of strategy performance.

`research-matrix-*.md` is the ablation/operating-mode report. A `not_run` scenario is an explicit missing-input result, never a zero-return result.
